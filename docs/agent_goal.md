# Dust Heist Agent Goal

Edit this file to tell the build agent what to improve next.

## Active Goal

Make Dust Heist feel like the most stylish and intense western arena survival game: amazing visuals, memorable characters with story, and a challenging arena that rewards skill.

## Current Focus

1. Add named duelist variants with story flavor and different combat pressure.
2. Improve arena challenge with readable obstacles, hazards, or wave modifiers.
3. Add stronger visual spectacle through lighting, VFX, camera, and HUD polish.

## Definition Of Amazing

- A new player understands the controls and goal quickly.
- A returning player has unlocks, named rivals, and escalating reasons to replay.
- Combat feels fast, lethal, readable, and fair.
- The arena looks like a real Dust Heist place, not a blank rectangle.
- Every enemy type has a recognizable role and silhouette.
- Every boss or duelist has a name, personality cue, and mechanical identity.

## Constraints

- Keep the game playable in Godot 4.
- Preserve the old-west Dust Heist art direction in `docs/art_bible.md`.
- Prefer one focused, verified improvement per agent pass.
- Validate the project before declaring a pass complete.

## Ideas Queue

- Add a duelist roster: The Black Sash, Mercy Vale, Colt Wren, Reverend Ash, and June Blackglass.
- Add duel intro lines and victory/defeat flavor.
- Add barricades, crates, lantern hazards, canyon rocks, or moving dust walls.
- Add wave modifiers such as Crossfire, Sandstorm, Gold Rush, Last Stand, or High Noon.
- Add combo ranks and end-of-run grading.
- Add ammo, reload timing, and weapon upgrade identities.
- Add music stingers, gunshots, slash sounds, and hit-freeze tuning.
